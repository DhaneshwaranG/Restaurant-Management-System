# Restaurant Management System

A full-stack restaurant management application built using React, Spring Boot, and MySQL.

## Features

- User Registration & Login
- JWT Authentication
- Category Management
- Menu Management
- Cart Functionality
- Order Placement
- Order History
- Admin Dashboard
- Revenue Analytics
- Order Status Tracking

## Tech Stack

### Frontend
- React
- Axios
- React Router

### Backend
- Spring Boot
- Spring Security
- JWT
- JPA / Hibernate

### Database
- MySQL
